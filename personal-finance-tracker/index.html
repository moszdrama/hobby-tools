Setup Guide & Deployment Instructions

1. Copy the code in Code.gs to your Google Sheet's Apps Script editor (Extensions > Apps Script).

2. In the Apps Script console, run the setupSheets function first to create transaction and category tables, and then run setupSecret to save the secret token.

3. Deploy as a Web App: click Deploy > New deployment, select Web app, configure execute as Me, and access to Anyone.

4. Authorize access, copy the Web App URL, paste it here, input your token, and connect.
