เชื่อมต่อ Google Sheets เพื่อเริ่มต้นใช้งาน

คู่มือติดตั้งและการ Deploy

1. คัดลอกโค้ดจาก Code.gs ไปวางใน Apps Script Editor ของ Google Sheet (Extensions > Apps Script)

2. ใน Apps Script ให้รันฟังก์ชัน setupSheets ก่อน เพื่อสร้างตาราง จากนั้นรัน setupSecret เพื่อบันทึก token ลับ

3. Deploy เป็น Web App: คลิก Deploy > New deployment เลือก Web app ตั้งค่า Execute as: Me และ Access: Anyone

4. อนุญาตสิทธิ์ คัดลอก URL ของ Web App มาวางในช่องด้านบน ใส่ token แล้วกดเชื่อมต่อ
